Tentang Sistem: 

Sistem PMB (Penerimaan Mahasiswa Baru) Online adalah aplikasi berbasis web yang digunakan untuk mempermudah proses pendaftaran calon mahasiswa secara digital. Sistem ini dirancang agar proses pendaftaran menjadi lebih efisien, cepat, dan terorganisir, baik untuk calon mahasiswa maupun pihak admin.

Fitur utama dalam sistem ini meliputi:

Pendaftaran akun calon mahasiswa
Pengisian formulir pendaftaran online
Upload dokumen persyaratan
Manajemen data pendaftar oleh admin
Monitoring status pendaftaran
Dashboard admin
Teknologi yang Digunakan

Sistem ini dibangun menggunakan teknologi berikut:

Laravel (Backend Framework)
MySQL (Database)
Tailwind CSS (Frontend Styling)
JavaScript (Interaktivitas)
